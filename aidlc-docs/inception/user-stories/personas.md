# 페르소나 정의 (Personas)

## Persona 1: 일반 고객 (Customer)

| 항목 | 내용 |
|---|---|
| **이름** | 김지수 |
| **역할** | 식당을 방문한 일반 고객 |
| **나이대** | 20~50대 |
| **기술 수준** | 스마트폰/태블릿 기본 사용 가능 |
| **목표** | 대기 없이 빠르게 원하는 메뉴를 주문하고, 주문 상태를 확인하고 싶다 |
| **불편사항** | 직원을 불러야 하는 번거로움, 주문이 제대로 들어갔는지 불안함 |
| **사용 환경** | 테이블에 설치된 태블릿 (터치 화면) |

**주요 시나리오:**
- 테이블에 앉자마자 태블릿을 켜면 바로 메뉴가 보임
- 카테고리를 탐색하며 원하는 메뉴를 고르고 장바구니에 담음
- 주문 확정 후 주문 번호를 확인하고 주문 내역에서 상태를 추적함

## Persona 2: 매장 관리자 (Store Admin)

| 항목 | 내용 |
|---|---|
| **이름** | 박철수 |
| **역할** | 식당 운영 담당자 (사장 또는 홀 직원) |
| **나이대** | 30~60대 |
| **기술 수준** | 기본적인 웹 브라우저 사용 가능 |
| **목표** | 테이블별 주문을 한눈에 파악하고, 신속하게 처리 상태를 업데이트하며, 정산이 끝난 테이블을 깔끔하게 마무리하고 싶다 |
| **불편사항** | 여러 테이블 주문을 동시에 관리하기 어려움, 구두 주문 실수 발생 |
| **사용 환경** | 카운터 또는 주방 근처의 PC/태블릿 |

**주요 시나리오:**
- 관리자 화면을 열면 모든 테이블의 현재 주문 현황이 그리드로 보임
- 신규 주문이 들어오면 시각적으로 강조되어 바로 확인 가능
- 음식 준비 완료 시 주문 상태를 변경하고, 고객이 계산 후 테이블 이용 완료 처리
- 잘못 들어온 주문은 직권으로 삭제
